CitySounds

An Interactive Musical Globe

Technologies Used:

Java (SpringBoot Framework), Javascript, HTML, CSS

CitySounds is a full-stack Java Spring Boot application that visualizes the relationship between geography and music. The app displays a 3D interactive globe where each city is connected to others through intelligent musical relationships. This application combines data persistence, external API integration, 3D graphics, and RESTful architecture into one project.
